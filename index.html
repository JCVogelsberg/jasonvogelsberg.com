SCIC Vendor Database

A single page app I built for the Sport Compact & Import Club at my alma mater, ERAU. It allows users to recommend shops, mechanics and companies with whom they have had a good experience. It uses Ruby on Rails and Ember.js.

Oregon Humane Society Pet Finder

App that allows the owners of lost pets to report them missing and post photos. People can also post stray animals they have found, and lost pet owners can email the finders directly. Built as part of my final group project at Epicodus.

Breaker-Breaker!

An online app that allows you to send text messages to a phone from your computer. It is built with Ruby on Rails, using the Twilio API for text messaging. The front end is styled with Twitter Bootstrap.



Acura EL: America's Rarest Acura

A Wordpress site I created documenting the Acura EL, a car sold only in the Canadian market, and rarely seen in the US.

Bubble-Loco!

A simple game I created using HTML5, CSS3, and jQuery. A random color is selected as a target, and the object is to click all the bubbles of that color, while they randomly change size and position.

My Story

I'm J. Vogelsberg, a web developer living in Portland, Oregon. Originally from Louisville, Kentucky, I spent most of the last decade living in Florida. Enjoying a wide range of interests, I started out as a journalism major, then switched to an Aviation Maintenance degree, graduating from Embry-Riddle Aeronautical University in 2009. I worked in Light Sport aviation, helicopters, and Part-161 flight training operations before deciding I wanted a new technical challenge. This brought me first to web development, then to Oregon.


When I'm not messing around in front of my MacBook, I enjoy building and rallying Hondas and Subarus, reading, archery, drinking ales, drinking tea, artisanal ballistics, writing, studying history, and sailing.

Education

In 2014, I graduated from Epicodus, an intensive, four-month web development bootcamp, where I studied HTML, CSS, Javascript/jQuery, Ruby/Rails, PostgreSQL, and Ember.js. Test-driven development was practiced throughout, using Mocha/Chai, Rspec, and Capybara. All work was done in pairs or small groups, and Github was used for version control.

Since leaving Epicodus, I have completed the Code Oregon/Team Treehouse Front End Web Development track, as well as courses in Bootstrap, Foundation, and WordPress.
